# NaiveProxy + OlcRTC Installer for DietPi & Linux

Этот репозиторий предоставляет автоматизированное решение для развертывания связки **NaiveProxy** (через Caddy) и **OlcRTC** на Linux-серверах. Скрипт специально оптимизирован для работы на **DietPi**, но также отлично работает на **Debian**, **Ubuntu** и других популярных дистрибутивах.

## Почему эта связка?
- **NaiveProxy**: Использует стек сетевых технологий Chrome для обхода цензуры, что делает его крайне трудным для обнаружения.
- **OlcRTC**: Позволяет туннелировать трафик через легальные WebRTC сервисы (например, конференции), что обеспечивает дополнительный уровень скрытности в условиях жестких ограничений.

## Особенности
- **Автоматизация**: Полный цикл установки от зависимостей до системных сервисов.
- **Интерактивное меню**: Удобная настройка без необходимости ручного редактирования конфигов.
- **Стабильность**: Использование systemd для автоматического перезапуска сервисов при сбоях.
- **Безопасность**: Автоматическая генерация ключей и настройка брандмауэра.
- **Легкость**: Минимум конфликтов и простая структура проекта.

## Быстрый старт

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/your-repo/installer.git
   cd installer
   ```

2. **Запустите меню:**
   ```bash
   sudo ./menu.sh
   ```

3. **Следуйте инструкциям в меню:**
   - Выберите пункт **1** для установки всех компонентов.
   - Выберите пункты **2** и **3** для настройки прокси и туннеля.
   - Запустите сервисы через пункт **4**.

## Структура проекта
- `menu.sh` — Главный скрипт управления.
- `scripts/install.sh` — Скрипт установки компонентов и зависимостей.
- `scripts/utils.sh` — Вспомогательные функции (детектирование ОС, порта, брандмауэра).
- `configs/` — Шаблоны конфигурационных файлов.
- `services/` — Шаблоны systemd сервисов.

## Совместимость
- **OS**: DietPi, Debian, Ubuntu, Raspbian, Arch (частично).
- **Arch**: x86_64 (amd64), ARM64, ARMv7.

## Поддержка OlcRTC
Проект OlcRTC находится в стадии беты. Для получения актуальной информации посетите [официальный репозиторий](https://github.com/openlibrecommunity/olcrtc).
